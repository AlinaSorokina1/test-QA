Тест-кейсы для формы логина

Позитивные тест-кейсы:

Тест-кейс 1: Корректный ввод email-адреса и пароля

Шаги: 
1. Ввести корректный email-адрес (например, 'user@example.com') в поле 'Username';
2. Ввести корректный пароль, содержащий не менее 8 символов (например, 'Password1') в поле 'Password'.

Ожидаемый результат: кнопка 'Login' активна.

Тест-кейс 2: Корректный ввод email-адреса и пароля с пробелами по краям

Шаги: 
1. Ввести корректный email-адрес с пробелами по краям (например, ' user@example.com ') в поле 'Username';
2. Ввести корректный пароль, содержащий не менее 8 символов, с пробелами по краям (например, ' Password1 ') в поле 'Password'.

Ожидаемый результат: кнопка 'Login' активна, пробелы удаляются.

Негативные тест-кейсы:

Тест-кейс 3: Пустое поле 'Username'

Шаги:
1. Оставить поле 'Username' пустым;
2. Ввести корректный пароль, содержащий не менее 8 символов (например, 'Password1') в поле 'Password'.

Ожидаемый результат: кнопка 'Login' неактивна, появляется сообщение об ошибке для поля 'Username'.

Тест-кейс 4: Неверный формат email-адреса в поле 'Username'

Шаги:
1. Ввести некорректный email-адрес (например, 'userexample.com') в поле 'Username';
2. Ввести корректный пароль, содержащий не менее 8 символов (например, 'Password1') в поле 'Password'.

Ожидаемый результат: кнопка 'Login' неактивна, появляется сообщение об ошибке для поля 'Username'.

Тест-кейс 5: Пустое поле 'Password'

Шаги:
1. Ввести корректный email-адрес (например, 'user@example.com') в поле 'Username';
2. Оставить поле 'Password' пустым.

Ожидаемый результат: кнопка 'Login' неактивна, появляется сообщение об ошибке для поля 'Password'.

Тест-кейс 6: Пароль содержит менее 8 символов

Шаги:
1. Ввести корректный email-адрес (например, 'user@example.com') в поле 'Username';
2. Ввести пароль менее 8 символов (например, 'Pwd1') в поле 'Password'.

Ожидаемый результат: кнопка 'Login' неактивна, появляется сообщение об ошибке для поля 'Password'.

Тест-кейс 7: Проверка отключения кнопки 'Login' при некорректном вводе

Шаги:
1. Ввести некорректный email-адрес (например, 'userexample.com') в поле 'Username';
2. Ввести пароль менее 8 символов (например, 'Pwd1') в поле 'Password'.

Ожидаемый результат: Кнопка 'Login' неактивна, появляется сообщение об ошибках для обоих полей.

Тест-кейс 8: Проверка отключения кнопки 'Login' при пустых полях 'Username' и 'Password'.

Шаги:
1.  Оставить поле 'Username' пустым;
2. Оставить поле 'Password' пустым.

Ожидаемый результат: Кнопка 'Login' неактивна, появляется сообщение об ошибках для обоих полей.
